# **App Name**: Code Scroller

## Core Features:

- Content Feed: Scrollable feed for browsing educational content.
- Search & Filter: Search and filtering functionality to find topics quickly.
- Saved Content: User's can save posts for later reviewing and access via a dedicated saved section.
- Content Creation: Content upload system, so authorized teachers can create posts on various topics
- Topic Simplification: AI-powered tool to simplify complex paragraphs. User's can request the system to use simple terms for easier understanding.

## Style Guidelines:

- Primary color: Deep dark blue (#1A237E), embodying professionalism and knowledge.
- Background color: Light off-white (#F5F5F5) to ensure readability and a clean interface in the light mode. To ensure consistency with the initial request for a dark and white color scheme, use dark blues and light grays for the Dark Mode variation of the UI.
- Accent color: Electric Blue (#7DF9FF) for interactive elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use modern and minimalist icons to represent subjects and functionalities.
- Clean and structured layout, similar to Instagram's, with a focus on readability.
- Subtle animations and transitions to enhance user experience without being distracting.
